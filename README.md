# security-writeups

This repository hosts write-ups on cybersecurity.

Each of them was written based on projects or labs i worked on during my free time.

These write-ups:

  - Do not contain exploitation of unknown or unpatched vulnerabilities
  - Can be viewed as a means to acquire knowledge to make your system more secure
  - Should be replicated only in an environment where you have explicit permission to perform these tests

[My WebApp Security Assessment](https://github.com/CyberGitLabs/security-writeups/blob/main/WebApp%20Security%20Assessment.md) explains how i tried to reach a good level of security during the deployment of my custom web application.

I hope you'll enjoy them.
